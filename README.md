# Coffee_shop

A UI representation of a Coffee Shop ☕️

A mobile application designed for a coffee business. It showcases :
- The coffee shop’s products,
- Allows users to customize a selected coffee (size, options),
- Add it to favorites or the cart,
- And manage the cart before placing an order.

You can download the .apk by checking the releases of this repository ! (Coming soon...)
